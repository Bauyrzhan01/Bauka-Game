# Bauka Game — Counter-Strike (Mobile)

Мобильная браузерная копия Counter-Strike с **онлайн-режимом** для игры с друзьями.

## Только телефон

Игра работает **только на смартфоне** (ландшафт). На ПК показывается заглушка.

## Запуск сервера

```bash
npm install
npm start
```

Открой с телефона (в той же Wi‑Fi сети):

```
http://IP-ТВОЕГО-ПК:3000
```

Узнать IP: `ipconfig` (Windows) → IPv4.

## Режимы

### Одиночная
- Боты CT, бомба, магазин, раунды

### Онлайн с друзьями
1. **Онлайн с друзьями** → **Создать комнату**
2. Отправь **код комнаты** друзьям
3. Друзья: **Войти** + код
4. Хост жмёт **Начать игру** (минимум 2 игрока)
5. T vs CT — убивайте врагов другой команды

## Управление (телефон)

| Элемент | Действие |
|---------|----------|
| Левый джойстик | Движение |
| Правая половина экрана | Прицел |
| ОГОНЬ | Стрелять |
| R | Перезарядка |
| B | Магазин (одиночная) |
| E | Бомба (одиночная) |
| ☰ | Меню паузы |
| ESC | Пауза (если есть клавиатура) |
