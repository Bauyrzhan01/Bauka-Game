# Bauka Game — Counter-Strike (Mobile)

Мобильная браузерная копия Counter-Strike. **Сервер — твой ноутбук.**

## Быстрый старт (ноутбук = сервер)

### 1. Запусти сервер на ноутбуке

```bash
npm install
npm start
```

В терминале появятся ссылки, например:
```
Для друзей: http://192.168.1.5:3000
```

### 2. Хост (ты)

1. На телефоне открой `http://192.168.1.5:3000` (твой IP из терминала)
2. **Онлайн с друзьями** → **Создать комнату**
3. Отправь друзьям:
   - ссылку `http://192.168.1.5:3000`
   - код комнаты `ABCD`

### 3. Друзья

1. Тот же Wi‑Fi что и ноутбук
2. Открывают ссылку на телефоне
3. **Онлайн** → вводят код → **Войти**
4. Ты жмёшь **Начать игру**

> **Важно:** ноутбук не засыпает, `npm start` работает, телефоны в **одной Wi‑Fi**.

---

## Если друг не в той же Wi‑Fi

Запусти туннель (ноутбук):

```bash
npm run tunnel
```

Появится ссылка типа `https://xxxx.loca.lt` — отправь её друзьям.
